# メトロノームクリック音

このディレクトリには、メトロノームのクリック音ファイルを配置します。

## ファイル仕様

| ファイル名 | 用途 | 推奨仕様 |
|-----------|------|---------|
| `click-normal.wav` | 通常拍のクリック音 | 44.1kHz, 16bit, モノラル |
| `click-accent.wav` | アクセント拍のクリック音 | 44.1kHz, 16bit, モノラル |

## 注意事項

- ファイルは50ms以下の短い音が推奨
- 十分な音量レベルでノーマライズ済みであること
- 著作権フリーまたはライセンス取得済みの音源を使用すること

## 現在の実装

音声ファイルが存在しない場合、オシレーターによる合成音が使用されます：
- 通常拍: 800Hz サイン波（50ms、指数関数的減衰）
- アクセント拍: 1000Hz サイン波（50ms、指数関数的減衰）
